Common Issues & Solutions

  • File upload not working: Check internet and file type.
  • Document not showing in progress: Refresh or check Submission History.
  • Unable to login: Reset password or contact support.

Tips for Smooth Submission

  • Use clear file names
  • Check file size limits
  • Select correct department
  • Keep copies of submissions

Step-by-Step Submission Guide

  1. Log in to the student portal
  2. Click "Submit Documents"
  3. Select document type and department
  4. Upload files
  5. Add notes (if needed)
  6. Click Submit
  7. Check Submission History

System Notices & Updates
